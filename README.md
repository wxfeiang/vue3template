# 项目配置

### 安装项目依赖
 
  ```
  pnpm i 

  定的依赖只有本地开发环境或者测试的时候才会安装 devDependencies
  生产环境下只安装 dependencies 中指定的依赖。

  ```
### 运行项目
 
  ```
  pnpm   run  dev 

  ```
### 打包
 
  ```
 pnpm  run  build 

  ```
